Elms Sans Variable Font
=======================

This download contains Elms Sans as both variable fonts and static fonts.

Elms Sans is a variable font with this axis:
  wght

This means all the styles are contained in these files:
  ElmsSans-VariableFont_wght.ttf
  ElmsSans-Italic-VariableFont_wght.ttf
